# chatgpt-prompt

## Environment
- vscode
  - plugins: Jupyter

- python3 (version >= 3.9)
```
python -m venv ./venv
.\venv\Scripts\activate # or: source ./venv/bin/activate
```

In VS Code, select Python interpreter to the "venv".

Run in git bash:
```
python -m pip install --upgrade pip
pip install -r requirements.txt
```
